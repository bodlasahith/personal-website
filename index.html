Aerial Drone WiFi Network

×

Aerial Drone WiFi Network

Aug 2024 - Dec 2024

Designed a mesh network of drones that provide WiFi coverage to remote areas without internet access. The drones are equipped with Raspberry Pi 4s and ESP32 microcontrollers that act as access points and relay stations. The Pis are connected to a central base station that manages the network and provides internet access via a 5G LTE connection. The ESP32s communicate over ESP-NOW that allows for low-latency communication between drones by directly pinging each other's MAC addresses. This project has the potential to revolutionize connectivity in underserved regions, enabling access to education, healthcare, and emergency services. It can also be deployed in disaster-stricken areas to quickly restore communication networks, significantly aiding in relief and rescue operations.

Object Detection System for the Visually Impaired

×

Object Detection System for the Visually Impaired

May 2021 - Aug 2021

This device provides the visually impaired with knowledge of their surroundings. Though the blind generally use canes to map out their path a few steps in front of themselves, this device corroborates some information gathered from touch using hearing. The system is built using an Arduino, breadboard, ultrasonic sensors, and a speaker. The setup was mounted to a lightweight cardboard wearable cutout to sit on the user's nose similar to a pair of glasses to simulate the experience. The ultrasonic sensors detect obstacles in the path of the user and relay this information to the Arduino, which then processes the data and sends auditory signals through the speaker to alert the user of nearby objects. This setup is compact, cost-effective, and easy to use, making it a practical solution for enhancing the mobility and independence of visually impaired individuals. By providing real-time feedback on their surroundings, this device significantly improves their ability to navigate safely and confidently in various environments.
