Asynchronous Distributed System

Aug 2024 - Dec 2024

Developed a distributed system with 10 VMs that allows users to run multiple tasks concurrently on a network of computers. The system uses a SWIM-based architecture architecture to distribute tasks to worker nodes and collect results as well as monitor and reboot machines that fail. A robust mapreduce-like architecture was implemented to handle task distribution and simulated an Apache Spark-like stream processing algorithm.

Aerial Drone WiFi Network

Aug 2024 - Dec 2024

Designed a mesh network of drones that provide WiFi coverage to remote areas without internet access. The drones are equipped with Raspberry Pi 4s and ESP32 microcontrollers that act as access points and relay stations. The Pis are connected to a central base station that manages the network and provides internet access via a 5G LTE connection. The ESP32s communicate over ESP-NOW that allows for low-latency communication between drones by directly pinging each other's MAC addresses.
